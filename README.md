# F-channel-Front


## Overview

2023年度鳥羽商船ハッカソン出展作品のフロントエンド用リポジトリ
[バックエンド](https://github.com/GoRuGoo/F-channel-Back)

## Requirement

### OS

- MacOS Ventura 13.0(動作確認済)

### Library

- Next.js
- Tailwind CSS

## Installation

1. Clone this repository

```
git clone git@github.com:GoRuGoo/F-channel-Front.git
```

2. Change Directory

```
cd F-channel-Front
```



## Usage
1. Install Library
```
npm install
```
2. Run locally
```
npm run dev
```
